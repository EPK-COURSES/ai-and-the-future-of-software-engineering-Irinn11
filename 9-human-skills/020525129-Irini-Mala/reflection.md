# Personal Reflection

Before researching this topic, I believed that AI might eventually replace a large percentage of software engineering jobs. After completing this research, my opinion changed significantly.

I realized that software engineering is not only about writing code. Developers also communicate with clients, work in teams, solve complex problems, make ethical decisions, and design systems that meet human needs. AI can help automate technical tasks, but it still lacks human understanding, emotional intelligence, and creativity.

The most surprising thing for me was how quickly AI tools are improving. Tools like ChatGPT and GitHub Copilot can already generate useful code and assist with debugging. However, I also discovered many limitations such as incorrect outputs, security risks, and overdependence on AI-generated solutions.

One concern I have is that some future developers may rely too much on AI without fully understanding programming concepts. This could reduce critical thinking skills and create engineers who can use tools but cannot solve problems independently.

At the same time, I also see many opportunities. AI can reduce repetitive work and allow engineers to focus on more creative and meaningful tasks. Developers may spend less time writing basic code and more time designing systems, solving business problems, and improving user experiences.

I believe communication and adaptability will become even more important in the future. Engineers who can learn quickly and work effectively with others will remain valuable even as technology changes.

Personally, I plan to adapt by continuing to improve both my technical and human skills. I want to strengthen my understanding of software engineering fundamentals while also learning how to use AI tools responsibly. I also want to improve my communication, teamwork, and problem-solving abilities because these skills cannot easily be automated.

Overall, this research helped me understand that AI should not only be seen as a threat. Instead, it can become a powerful tool that supports software engineers if used correctly. The future will likely belong to engineers who know how to combine human intelligence with AI assistance effectively.
